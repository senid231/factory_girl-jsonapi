# FactoryGirl::Jsonapi

Add ability to generate JSON API request/response body with FactoryGirl

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'factory_girl_json_api'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install factory_girl_json_api
